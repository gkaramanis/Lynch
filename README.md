
# Lynch

<!-- badges: start -->

Color palettes from David Lynch films.

## Installation

You can install the released version of Lynch from [CRAN](https://CRAN.R-project.org) with:

``` r
install.packages("Lynch")
```

## Example

lynch_palette("twin_peaks")
lynch_palette("blue_velvet", 3)

``` r
library(Lynch)
## basic example code
```
